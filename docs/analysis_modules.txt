请为我编写一个主要用于组学数据分析的R语言程序包。

对于这个组学数据分析R脚本程序，其接受的数据输入主要为csv表格文件，所需要接受的表格数据文件有：

1. 表达矩阵csv文件：行为feature表达数据，列为样本数据。第一行为列标题即样本ID，第一列为feature ID
2. 样本元数据文件：具有下面的三列必须字段：ID为表达矩阵中的样本ID，sample_name为进行绘图，用于论文中展示的样本显示标签文本，sample_info为样本的分组标签，其他的列为对应的样本ID的补充元数据
3. feature注释文件：具有下面的4列必须字段：ID为feature的ID，ID列与feature表达矩阵csv文件中的第一列feature ID对应。name为feature的英文俗名，type为feature的类别，可能的类别标签有（会存在不同别名） ：gene(转录组),rna（转录组）,protein（蛋白质组）,metabolite（代谢组）,lipid（脂质组）,organism（微生物组），bacterial(微生物组)， taxonomy（微生物组）。以及kegg列，可以用于分子类别的feature做KEGG通路相关的分析。可能的可选的列会有pfam，主要针对转录组或者蛋白质组中的分子，为pfam结构域的集合，使用分号作为分隔符；  可能的可选的列还有family，主要为对应的feature的分类家族或者ontology信息，例如代谢物的化学结构分类，蛋白质家族名称等分类标签信息

基于以上的输入文件描述，编写的R语言程序包中的函数模块为：

+ 缺失值过滤：按照所有分组中缺失比例超过百分比阈值后删除，任意一个分组中缺失比例没有超过阈值就保留；或者按照总体样本中缺失比例超过百分比阈值后删除
+ 缺失值填充： 按照feature最小阳性值一半做填充；或者KNN填充
+ 数据归一化： 按照样本总和做相对丰度归一化
+ 数据缩放： 按照feature做中位数数据缩放
+ QCQA： 计算QC分组样本的变异度，或者PCA无监督分析评估组内差异与组间差异的显著性
+ PCA以及PCA得分图
+ PLSDA以及PLSDA得分图，计算VIP
+ OPLSDA以及OPLSDA得分图，计算VIP
+ F检验整体差异分析
+ 多因素ANOVA整体差异分析
+ Limma差异，多种策略：按照pvalue+logFC差异；或者pvalue+VIP差异；或者pvalue筛选显著性，然后按照logFC绝对值降序排序后取top N得到差异性
+ 火山图，需要显示top 5差异的feature name
+ 文氏图
+ upset图
+ 热图，行一般为feature维度，列一般为样本维度。列一般按照样本分组做排序，feature维度行需要做层次聚类树，假若存在family信息，还需要使用颜色块显示feature的family
+ Fisher富集检验
+ GSVA
+ 构建WGCNA共表达模块
+ WGCNA生物性状关联与线性回归：在WGCNA共表达模块的基础上，计算共表达模块与生物性状的关联性，feature与生物性状的关联性，以及feature与生物学性状的线性回归
+ CMeans模糊聚类
+ bnlearn，对于时间序列数据，构建调控网络
+ PLS-PM，对于多组学或者单组学，可以以family或者kegg通路作为潜变量，计算潜变量之间的调控网络 
+ 随机森林+shap解释，针对样本进行分组预测模型的建立
